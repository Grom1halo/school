Разберу каждую функцию с конкретными числами из примера M=4, t=1:



\---



\### `weight(x)` — вес Хэмминга

\*\*Вход:\*\* одно целое число — двоичное слово

```

weight(7)   // 7 = 00111 → считаем единицы → вернёт 3

weight(25)  // 25 = 11001 → вернёт 3

weight(0)   // 00000 → вернёт 0

```



\---



\### `hammingDist(a, b)` — расстояние Хэмминга

\*\*Вход:\*\* два целых числа — два кодовых слова

```

hammingDist(0, 7)   // 00000 и 00111 → XOR = 00111 → weight(7) = 3

hammingDist(7, 25)  // 00111 и 11001 → XOR = 11110 → weight(30) = 4

hammingDist(7, 7)   // одинаковые → 0

```



\---



\### `minCodeDist(cw)` — минимальное расстояние кода

\*\*Вход:\*\* вектор из M целых чисел — все разрешённые слова

```

minCodeDist({0, 7, 25, 30})

// 0=00000, 7=00111, 25=11001, 30=11110

// перебирает все пары: (0,7)=3, (0,25)=3, (0,30)=4, (7,25)=4, (7,30)=3, (25,30)=3

// вернёт 3

```



\---



\### `findAllowed(n, M, dMin, result)` — жадный отбор

\*\*Вход:\*\* n=5, M=4, dMin=3, пустой вектор result

```

findAllowed(5, 4, 3, result)

// перебирает 0,1,2,...,31

// берёт 0  (00000) — первое всегда

// 1 (00001): d(1,0)=1 < 3 — пропуск

// ...

// 7 (00111): d(7,0)=3 ✓ — берёт

// ...

// 25 (11001): d(25,0)=3 ✓, d(25,7)=4 ✓ — берёт

// 30 (11110): d(30,0)=4 ✓, d(30,7)=3 ✓, d(30,25)=3 ✓ — берёт

// result = {0, 7, 25, 30}, вернёт true

```



\---



\### `findN(M, dMin)` — поиск длины кода

\*\*Вход:\*\* M=4, dMin=3

```

findN(4, 3)

// n = ceil(log2(4)) = 2

// findAllowed(2, 4, 3) → false  (всего 4 слова, не хватает)

// n=3 → false

// n=4 → false

// n=5 → true ✓

// вернёт 5

```



\---



\### `printBin(w, n)` — вывод в двоичном виде

\*\*Вход:\*\* w=25, n=5

```

printBin(25, 5)  // 25 = 11001 → выведет "11001"

printBin(0, 5)   // выведет "00000"

printBin(7, 5)   // выведет "00111"

```



\---



\### `readInt(prompt, lo, hi)` — безопасный ввод

\*\*Вход:\*\* строка-подсказка, lo=2, hi=512

```

readInt("Введите M: ", 2, 512)

// пользователь вводит "abc" → ошибка, повтор

// пользователь вводит "1"   → 1 < 2, ошибка, повтор

// пользователь вводит "4"   → ✓, вернёт 4

```



\---



\### `injectErrors(word, n, errCount, rng)` — внесение ошибок

\*\*Вход:\*\* word=7 (00111), n=5, errCount=1, rng=генератор

```

injectErrors(7, 5, 1, rng)

// pos = {0,1,2,3,4}, после shuffle например {3,1,4,0,2}

// инвертирует бит на позиции 3

// 00111 XOR 01000 = 01111 → вернёт 15

```



\---



\### `decode(received, allowed, M, t, isAllowedTable)` — декодирование

\*\*Вход:\*\* received=15 (01111), allowed={0,7,25,30}, M=4, t=1, таблица

```

decode(15, {0,7,25,30}, 4, 1, table)

// 15 не в таблице разрешённых

// d(15, 0)  = 4

// d(15, 7)  = 1  ← минимум, один такой

// d(15, 25) = 4

// d(15, 30) = 3

// bestDist=1 ≤ t=1 → corrected=true, data=1

```
