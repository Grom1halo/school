# Лаба 6 — Гайд для одногруппников

## Тема: UDP аудиопоток на языке C

## Варианты

| Вариант | Задание |
|---------|---------|
| **1** | UDP пакеты, метка RAW ← этот файл |
| 2 | ZMQ подписка, метка RAW |
| 3 | UDP пакеты, метка Z08 (LZW сжатие) |
| 4 | ZMQ подписка, метка Z08 |
| 5 | UDP пакеты, метка Z16 |
| 6 | ZMQ подписка, метка Z16 |
| 7 | ZMQ подписка, метка Z08 + лог потерь пакетов |

**Если у тебя другой вариант — напиши Никите.**

---

## Что нужно поменять (вариант 1)

1. **ФИО и дата** — в отчёте
2. **Порт** — уточни у преподавателя на паре (по умолчанию 9000)

---

## Установка и компиляция

```bash
sudo apt install -y gcc

gcc ./lab6_receiver.c -o ./lab6_receiver.bin
```

---

## Тест локально

```bash
# Запускаем тестовый отправитель (из репозитория)
gcc ./lab6_sender.c -o lab6_sender.bin -lm

# Приёмник в фоне
./lab6_receiver.bin 9000 audio_out.raw &

# Отправляем 20 тестовых пакетов
./lab6_sender.bin 127.0.0.1 9000 20
```

---

## На паре (с сервером преподавателя)

```bash
# Запускаем приёмник на порт преподавателя
./lab6_receiver.bin <порт_преподавателя> audio_out.raw

# После получения данных воспроизводим
aplay -r 8000 -f U8 -c 1 audio_out.raw   # для R08
```
