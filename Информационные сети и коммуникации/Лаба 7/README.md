# Лаба 7 — Гайд для одногруппников

## Тема: ZMQ PUBLISH/SUBSCRIBE

## Варианты

- **Нечётные команды (1, 3, 5...)** — получают **перевод** сонета Шекспира (ключ `RU`)
- **Чётные команды (2, 4, 6...)** — получают **оригинал** на английском (ключ `EN`)

Ключ сжатого канала определяется на паре из кода сервера преподавателя.

---

## Что нужно поменять под себя

1. **ФИО и дата** — в отчёте
2. **Ключ подписки** — если чётная команда, поменяй в `lab7_subscriber.c`:
   ```c
   #define KEY_TEXT "EN"   // вместо "RU"
   #define KEY_COMP "ZE"   // уточни у преподавателя
   ```
3. **IP сервера** — на паре передать IP преподавателя как аргумент

---

## Установка зависимостей

```bash
sudo apt install -y libzmq3-dev zlib1g-dev gcc
```

---

## Компиляция

```bash
gcc ./lab7_subscriber.c -o ./lab7_subscriber.bin -lz -lzmq
```

---

## Тест локально (с тестовым издателем)

```bash
# Компилируем издателя
gcc ./lab7_publisher.c -o ./lab7_publisher.bin -lz -lzmq

# Подписчик в фоне
./lab7_subscriber.bin 127.0.0.1 5563 &

# Запускаем издателя
./lab7_publisher.bin
```

---

## На паре (с сервером преподавателя)

```bash
./lab7_subscriber.bin <IP_преподавателя> 5563
```
