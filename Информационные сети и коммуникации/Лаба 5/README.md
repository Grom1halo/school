# Лаба 5 — Гайд для одногруппников

## Тема: TCP сервер на языке C

## Варианты (у всех разные!)

| Вариант | Задание |
|---------|---------|
| **1** | Сервер аутентификации (Login/Password → время или отказ) ← этот файл |
| 2 | Сервер проверки времени (клиент шлёт UNIX time, сервер отвечает разницей) |
| 3 | Клиент получает UNIX time от сервера |
| 4 | HTTP-сервер, отдаёт HTML-страницу |
| 5 | HTTP-сервер, отдаёт текстовый файл |
| 6 | HTTP-сервер, отдаёт картинку |
| 7 | HTTP-сервер, отдаёт бинарный файл |
| 8 | Сервер с подсказкой времени, клиент шлёт UNIX time |

**Если у тебя другой вариант — напиши Никите, он скинет нужный код.**

---

## Что нужно поменять под себя (вариант 1)

1. **ФИО и дата** — в отчёте
2. **Логин/пароль** — в `lab5_server.c` строки `#define LOGIN` и `#define PASS` (если хочешь поменять)
3. **Порт** — по умолчанию 5555, можно изменить через `#define PORT`

---

## Компиляция и запуск

```bash
# Компиляция
gcc ./lab5_server.c -o ./lab5_server.bin

# Запуск сервера
./lab5_server.bin

# Тест в другом терминале
printf "admin\n123\n" | nc 127.0.0.1 5555
printf "hacker\nwrong\n" | nc 127.0.0.1 5555
```

---

## Захват трафика

```bash
sudo tcpdump -i lo tcp port 5555 -w dump.pcap &
# ... запускаешь тесты ...
sudo tcpdump -r dump.pcap -A > tcpdump.rep
```
