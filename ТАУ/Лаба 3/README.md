## Установка

### Python (Windows)
Скачай Python с https://python.org, при установке поставь галочку "Add to PATH".
Затем в PowerShell:
```powershell
pip install numpy matplotlib scipy
```

### Node.js (Windows)
Скачай с https://nodejs.org и установи.
Затем в PowerShell:
```powershell
npm install -g docx
```

## Запуск

```powershell
# 1. Запусти симуляцию (создаёт графики в папке plots/)
python lab3_sim.py

# 2. Сгенерируй Word-отчёт
set NODE_PATH=%APPDATA%\npm\node_modules
node gen_report.js
```

После этого в папке появится `lab3_report.docx`.

## Что делает симуляция

- Моделирует систему W₁(εp)·W₂(p)·W₃(p) при разных ε
- Исследует разделение движений: «быстрые» (W₁) и «медленные» (W₂·W₃)
- Строит фазовые портреты (x₁, x₂) при разных ε
- Вычисляет степень разделимости D = |Re(λ_fast)| / |λ_slow|
- Рассчитывает ε₀ из условия D = 10
