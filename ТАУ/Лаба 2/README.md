## Установка

### Python (Windows)
Скачай Python с https://python.org, при установке поставь галочку "Add to PATH".
Затем в PowerShell:
```powershell
pip install numpy matplotlib scipy
```

### Node.js (Windows)
Скачай с https://nodejs.org и установи.
Затем в PowerShell:
```powershell
npm install -g docx
```

## Запуск симуляции и генерации отчёта

```powershell
# 1. Запусти симуляцию (создаёт графики в папке plots/)
python lab2_sim.py

# 2. Сгенерируй Word-отчёт
set NODE_PATH=%APPDATA%\npm\node_modules
node gen_report.js
```

После этого в папке появится `lab2_report.docx`.

## Что делает симуляция

- Моделирует нелинейную систему с W_л(p) = K/(p·(T₁p+1)·(T₂p+1))
- Три типа нелинейностей: идеальное реле, реле с зоной нечувствительности, реле с гистерезисом
- Строит фазовые портреты, переходные характеристики
- Строит годограф Голдфарба для аналитического расчёта автоколебаний
