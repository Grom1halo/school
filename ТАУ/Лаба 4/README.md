## Установка

### Python (Windows)
Скачай Python с https://python.org, при установке поставь галочку "Add to PATH".
Затем в PowerShell:
```powershell
pip install numpy matplotlib scipy
```

### Node.js (Windows)
Скачай с https://nodejs.org и установи.
Затем в PowerShell:
```powershell
npm install -g docx
```

## Запуск

```powershell
# 1. Запусти симуляцию (создаёт графики в папке plots/)
python lab4_sim.py

# 2. Сгенерируй Word-отчёт
set NODE_PATH=%APPDATA%\npm\node_modules
node gen_report.js
```

После этого в папке появится `lab4_report.docx`.

## Что делает симуляция

- Объект управления: ÿ + a₂ẏ + a₁y² = bU − M(t) (нелинейный)
- Синтез регулятора методом локализации с дифференцирующим фильтром 2-го порядка
- Исследование влияния: коэффициента K, быстрого/медленного фильтра, возмущения M(t), ограничения U_max
- Все режимы моделируются методом Рунге-Кутта 4-го порядка
