# Курсовая работа ТАУ — Вариант 4.5 — Гайд для одногруппников

## Тема
Синтез ПИД-регулятора для системы стабилизации мощности резания процесса сквозного бесцентрового шлифования.

## Что нужно поменять под себя

1. **ФИО** — в `gen_report.js` найди строку `Кровин Н.И.` и замени на своё
2. **Вариант** — у каждого свои параметры объекта. В `sim.py` поменяй:
   - Параметры объекта (R1, T1, C1, C2, J, k1, k2, K3, T2 и др.)
   - Диапазон h (глубина резания)
   - Требования к качеству (tп, δ)

---

## Установка

### Python (Windows)
Скачай Python с https://python.org, при установке поставь галочку "Add to PATH".
Затем в PowerShell:
```powershell
pip install numpy matplotlib scipy
```

### Node.js (Windows)
Скачай с https://nodejs.org и установи.
Затем в PowerShell:
```powershell
npm install -g docx
```

---

## Запуск

```powershell
# 1. Запусти симуляцию (строит все графики, оптимизирует ПИД)
python sim.py

# 2. Сгенерируй Word-отчёт
set NODE_PATH=%APPDATA%\npm\node_modules
node gen_report.js
```

---

## Что делает симуляция

- Строит математическую модель привода и объекта управления
- Синтезирует ПИД-регулятор методом частотного подбора (scipy differential_evolution)
- Проверяет запасы устойчивости по АЧХ/ФЧХ (Bode-диаграмма)
- Исследует робастность при изменении параметра h (глубина резания)
- Строит переходные характеристики и ЛАХ/ЛФХ
